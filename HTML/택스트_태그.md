## 택스트 태그 정리

<br>

### ▶ 문단 (p태그)

- 문단 요소를 나타내는 태그 (자주 사용)

```
<p>안녕하세요.</p>
<p>공백이 생깁니다</p>
```

결과

```
안녕하세요.

공백이 생깁니다
```

<br>

### ▶ 제목 (h태그)

- 숫자와 함께 표현 (숫자가 작을 수록 제목이 크다.)

```
<h1>제목 태그는 h이다.</h1>
<h2>제목 태그는 h이다.</h2>
.
.
<h6>제목 태그는 h이다.</h6>
```

<br>

### ▶ 수평선 (hr태그)

- 단일태그

```
<p>수평선 태그는 hr이다.</p>
<hr>
<p>수평선 태그는 hr이다.</P>
```
