## 상대 단위

<br>

- px는 절대 길이 단위로 항상 동일한 값 유지, 가변성이 없다.



### em과 rem

- 박스에서 텍스트 크기 조정 시에 사용하는 상대 단위
- em은 부모 요소의 글꼴 크기, rem은 루트 요소의 크기

```
<div style="font-size: 20px;">
	<p>자식 요소의 1em은 20px!</p>
	<p>자식 요소의 2em은 40px!</p>
</div>
```



#### ※em으로 내, 외부 여백 크기 정할 때 자기 자신의 글자 크기 기준으로 함
