## 박스모델(Box-model)

<br>

- 브라우저가 요소를 렌더링 할 때, 각각의 요소는 디폴트 값으로 사각형 형태로 영역을 차지
- CSS는 박스 크기, 위치, 속성을 결정할 수 있다.



<br>

- 콘텐츠 영역: width, height
- 안쪽 여백 : padding
- 바깥쪽 여백 : margin
- 테두리 : border-width



#### ※ span 태그는 인라인 요소로 width, height 설정을 할 수는 없지만,

####  display:inline-block 속성을 이용하여 설정이 가능하다.
