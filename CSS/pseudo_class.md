## 의사클래스

<br>

- 선택자에 추가하는 키워드
- 요소가 특정 상태가 되었을 때 요소를 선택

```
선택자:의사클래스{
	속성명: 속성값;
}
```

```
h1:hover{
	color:red;
}
```

<br>

| 의사클래스  |                    의미                    |
| :---------: | :----------------------------------------: |
|    hover    |        마우스 커서가 요소에 있을 때        |
|   active    | 사용자가 요소를 활성화했다.(마우스로 클릭) |
|    focus    |             포커스를 받고 있다             |
|  disabled   |                비활성 상태                 |
| nth-child() |   형제관계에 있는 순서에 따라 요소 선택    |
