## background

<br>

- 콘텐츠의 배경을 정의(단축 속성)

| 하위 속성           | 역할                       |
| ------------------- | -------------------------- |
| background-color    | 배경 색 정의               |
| background-image    | 배경 이미지 정의           |
| background-position | 배경 이미지 초기 위치 정의 |
| background-size     | 배경 이미지 크기 정의      |
| background-repeat   | 배경 이미지 반복 정의      |
