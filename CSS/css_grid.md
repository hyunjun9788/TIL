### grid-template-columns, grid-template-rows

- 그리드 컨테이너의 트랙 중 열,행 트랙 내 아이템들의 크기를 지정할 수 있는 속성
- none: 기본값, 수치: 음수X

```
grid-template-columns:100px 1fr;
grid-template-row: 200px 1fr 1fr;
```



### gap

- 그리드 아이템 사이의 간격을 지정하는 속성.
- 행에서의 간격과 열에서의 간격을 똑같이 지정

```
gap:20px 10px;
```
